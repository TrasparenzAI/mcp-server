# MCP Server del progetto TrasparenzAI
## MCP Server - connettere le informazioni di TrasparenzAI con le applicazioni AI

[![Supported JVM Versions](https://img.shields.io/badge/JVM-25-brightgreen.svg?style=for-the-badge&logo=Java)](https://openjdk.java.net/install/)

Task Scheduler Service è parte della suite di servizi per la verifica delle informazioni sulla
Trasparenza dei siti web delle Pubbliche amministrazioni italiane.
 
## MCP Server TrasparenzAI

Questo progetto contiene un [MCP](https://modelcontextprotocol.io/docs/getting-started/intro) server
che espone come Tool alcune informazioni pubbliche della piattaforma TrasparenzAI.

## 👏 Come Contribuire 

E' possibile contribuire a questo progetto utilizzando le modalità standard della comunità opensource 
(issue + pull request) e siamo grati alla comunità per ogni contribuito a correggere bug e miglioramenti.
